Description:
Using neural networks (CNN) for image classification of fruits: grapes and apples

Files:
cnn_grapes - the main script containing the data preprocessing, model generation, training, evaluation
Readme.txt - this file

Folders:
analysis - overview of the used data and prediction statistics on selected images
dataset - apples and grapes images for training, testing and single predictions
img - graphs for accuracy and loss on training and validation(test) data
logs - IPython logs from model training and from predictions for single images (evaluation)
utils_parse_save - scripts and sample code to parse and plot the training logs generated during the model training phase
